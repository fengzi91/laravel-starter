

## Laravel 项目基础框架

## 包含定制
 - prettier 前端文件格式化
    > 使用 phpstorm 配置，Preferences -> Languages & Frameworks -> JavaScript -> Prettier -> Run for files -> 输入 `resources/{**/*,*}.{js,ts,jsx,tsx,vue,blade.php}` -> 选中 `On Save` 或者 `On code reformat`
